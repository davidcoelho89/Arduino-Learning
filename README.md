---------------------FEATURES--------------------------

=> Communication

- LCD
- Serial: USB / RS422 / RS485
- Bluetooth
- I2C
- SPI (HCS08)
- Radio
- Ethernet

=> Sensors

- ACC (Acelerometer)
- IR (infrared)
- LDR (photoresistor)
- LM35 (temperatura)
- RTC (Real Time Clock - DS1307)
- RFID (Radio-Frequency IDentification)

=> Actuators

- DC Motor
- Stepper Motor
- Servo Motor
- DipSwitch
- Button (Push / Toggle)
- Buzzer (RTTL)
- Led
